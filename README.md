# DripCalc

Калькулятор автополива для расчета расписания поливов растений.

## Возможности

- Расчет расписания поливов на основе светового режима
- Поддержка компенсированных капельниц
- Расчет расхода воды и объема бака
- Расчет даты следующего наполнения бака
- Создание напоминаний в iOS приложении "Напоминания"
- Адаптивный дизайн для мобильных устройств
- PWA поддержка

## Технологии

- React 19
- TypeScript
- Vite
- PWA (Progressive Web App)

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build

# Просмотр собранного проекта
npm run preview
```

## Использование

1. Настройте световой режим растений (часы света и время включения)
2. Установите частоту и количество поливов
3. При необходимости включите опцию "Компенсированные капельницы" и настройте параметры системы
4. Включите опцию "Расчитать расход" для расчета объема бака и даты наполнения
5. Просмотрите расписание поливов в таблице
